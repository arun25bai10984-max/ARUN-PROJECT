================================================================================
                    SIMPLE TO-DO LIST GUI APPLICATION
================================================================================

A beginner-friendly Python GUI application for managing daily tasks, 
built with tkinter.

================================================================================
DESCRIPTION
================================================================================

This is a simple yet functional to-do list application designed for Python 
beginners who want to learn GUI programming. The application provides an 
intuitive interface to add, delete, and manage tasks efficiently.

================================================================================
FEATURES
================================================================================

* Add Tasks: Type your task and press Enter or click the Add button
* Delete Tasks: Select any task and remove it with one click
* Clear All: Remove all tasks at once (with confirmation dialog)
* Scrollable List: Handle as many tasks as you need
* Clean Interface: User-friendly design with color-coded buttons

================================================================================
GETTING STARTED
================================================================================

Prerequisites:
--------------
* Python 3.x installed on your system
* tkinter (comes pre-installed with Python)

Installation:
-------------
1. Download the todo_app.py file
2. Navigate to the project directory
3. No additional packages required!

Running the Application:
------------------------
Open your command prompt or terminal and type:

    python todo_app.py

Or on some systems:

    python3 todo_app.py

================================================================================
HOW TO USE
================================================================================

1. Adding a Task:
   - Type your task in the text box at the top
   - Press Enter or click the "Add" button
   
2. Deleting a Task:
   - Click on the task you want to remove
   - Click the "Delete Task" button

3. Clearing All Tasks:
   - Click the "Clear All" button
   - Confirm the action in the popup dialog

================================================================================
TROUBLESHOOTING
================================================================================

Problem: "ModuleNotFoundError: No module named 'tkinter'"

Solution: 
* On Ubuntu/Debian: sudo apt-get install python3-tk
* On macOS: tkinter comes with Python
* On Windows: Reinstall Python with tkinter option checked
